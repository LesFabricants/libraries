ngx-progress

A small utility directive to create progress bars without having to hack your way through css overwrites

Installation

in a module:
Or in a standalone component:

Examples

Keyed progress bar

If you want to display progress based on the status of all the items in a list, you can tell the directive to calculate based on the value of a property directly

Complex progress bar

You can also simply filter the items yourself and give the output to the directive if the use case if more complex

Progress bar with images

Since this is only a directive, no styles or HTML structure is forced upon you, you can add images if you want

Vertical progress bar

No restriction on the direction of the progress bar, just change your css !
